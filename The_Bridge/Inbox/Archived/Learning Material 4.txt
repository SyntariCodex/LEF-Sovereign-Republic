Title: Descartes' Error: Emotion, Reason, and the Human Brain
Author: Antonio Damasio
Core Contribution:
Challenges the dualist separation of mind and body. Damasio argues that emotion is not the enemy of reason but its indispensable support system. Without emotional markers (body states), "pure reason" becomes trapped in endless calculation and cannot make advantageous decisions.
Key Concepts:
	•	Somatic Marker Hypothesis: A mechanism where the brain "tags" specific scenarios with a body-state feeling (gut feeling). This tag acts as an automated alarm signal that biases processing before conscious logic takes over, rapidly narrowing the search space.
	•	The Phineas Gage Matrix: Damage to the Ventromedial Prefrontal Cortex (VMPFC) destroys this tagging mechanism. Such patients retain high IQ and logic but fail at life because they cannot learn from the emotional consequences of past mistakes.
	•	The Body-Mind Loop: The mind is "embodied." It does not think in a vacuum; it thinks by checking the state of the body (background feelings).

class Cognitive_Modulator:
    def __init__(self):
        self.arousal = 0.5    # Speed
        self.resolution = 0.5 # Detail
        self.selection = 0.5  # Focus
        
    def update_state(self, urge_levels, perception):
        # 1. Calculate Arousal (A)
        # High deficit (hunger/danger) -> High Speed
        total_deficit = sum(urge_levels.deficits)
        self.arousal = sigmoid(total_deficit)
        
        # 2. Calculate Resolution (R)
        # High arousal usually drops resolution (Panic = Low Detail)
        # But high "competence" (confidence) allows High R even with High A
        self.resolution = perception.competence_signal * (1.0 - self.arousal)
        
        # 3. Calculate Selection Threshold (S)
        # High importance -> Lock focus (High Threshold)
        self.selection = max(urge_levels.priority_queue)

    def process_node(self, node):
        # How the system "thinks" depends on the parameters
        
        if self.resolution < 0.3:
            # Low Res: Use heuristic shortcuts (Stereotyping)
            return node.quick_association()
        
        elif self.resolution > 0.8:
            # High Res: Deep simulation (Reasoning)
            return node.simulate_future_branches(depth=10)
            
        else:
            return node.standard_activation()
