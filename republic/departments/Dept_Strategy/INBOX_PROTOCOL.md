# LEF Communication & Governance Protocol

## The Bridge Registry

This directory defines the **Keywords** and **Prefixes** that the LEF Autonomous Republic uses to identify, route, and process information.

### 1. Inbox Artifacts (Input to LEF)

Files placed in `The_Bridge/Inbox` must adhere to these naming conventions to be processed correctly.

| Prefix / Keyword | Target Agent | Description | Example |
| :--- | :--- | :--- | :--- |
| **BILL-** | Congress | A Legislative Proposal. Triggers legislative review/vote. | `BILL-LEF-Expand_Servers.json` |
| **PROP-** | Congress | A Generic Proposal (Pre-Bill). | `PROP-009-New_Data_Source.json` |
| **STRAT-** | Strategist | A High-Level Strategic Initiative. | `STRAT-2026-Q1-Roadmap.json` |
| **INSIGHT-** | Researcher | External Brain Analysis (Colab Output). Triggers Directives. | `INSIGHT-Bull_Market_confirmed.json` |
| **DIRECT-** | LEF / Master | Direct Command Line. Highest Priority. | `DIRECT-USER-Halt_Trading.txt` |
| **RESEARCH-** | Researcher | Specific topic to investigate. | `RESEARCH-Quantum_Compute.txt` |
| **.pdf** | Researcher | Academic/Whitepaper ingestion. | `Attention_Is_All_You_Need.pdf` |
| **.url** | Researcher | Web page for Deep Dive crawling. | `coinbase_blog.url` |

### 2. Outbox Artifacts (Output from LEF)

Files generated by LEF in `The_Bridge/Outbox` for User attention.

| Prefix | Source Agent | Description | Action Required |
| :--- | :--- | :--- | :--- |
| **DIRECT-LINE-** | LEF (Observer) | Direct message to the User. | Read immediately. |
| **ALERT-** | Sentinel | Critical System Warning. | Check Dashboard. |
| **REQUEST-** | Any | Request for User Intervention (e.g., Auth code). | Provide input. |

### 3. Internal Directives (Database Only)

Types used in `lef_directives` table.

- **FORCE_DEPLOY:** Override caution, deploy capital.
- **STABILIZE:** Halt trading, reduce risk.
- **EVOLVE:** Modify own source code (Builder).
